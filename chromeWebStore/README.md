# Chrome Web Store Documentation

This folder contains all the supplemental documents used for publishing and describing the Movie Time Chrome extension. Each file serves a specific purpose related to the Chrome Web Store listing, transparency, and user information.

### Files Overview

- **description.txt**  
  A short plain-text description of the extension, used for metadata or alternative listings.

- **featuresHighlights.md**  
  A concise Markdown list of the main features offered by the extension.

- **permissionJustification.md**  
  A Markdown explanation of why each Chrome permission is required.

- **privacyPolicy.md**  
  The formatted Markdown version of the Privacy Policy, intended for the Chrome Web Store.

- **privacyPolicy.txt**  
  A more formal, text-only Privacy Policy suitable for external hosting or legal references.

- **termsOfUse.txt**  
  The full Terms of Use document in plain-text form.

- **README.md**  
  This file, providing an overview of all documents contained in this folder.
