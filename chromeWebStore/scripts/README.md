# How to use

## Virtual Environment setup

1) crea venv
    ```bash
    python3 -m venv .venv
    ```

2) attiva venv
    ```bash
    source .venv/bin/activate
    ```

3) aggiorna pip
    ```bash
    python -m pip install --upgrade pip
    ```

4) installa dipendenze
    ```bash
    pip install -r requirements.txt
    ```

---

## Run the script

1) avvia la webapp
    ```bash
    streamlit run app.py
    ```
